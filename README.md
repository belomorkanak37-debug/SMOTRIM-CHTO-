# СМОТРИМ!ЧТО?

Мобильное веб-приложение для подбора фильмов, сериалов, мультфильмов и аниме. Интерфейс лежит в `public/index.html`, а `server.js` нужен для запуска на Render и безопасной работы с TMDB.

## Локальный запуск

1. Установите Node.js 20 или новее.
2. Создайте переменную окружения `TMDB_API_KEY` со своим ключом TMDB v3.
3. Запустите приложение:

```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000`.

Для PowerShell переменную можно задать так:

```powershell
$env:TMDB_API_KEY="ваш_ключ_tmdb"
npm start
```

## Деплой на Render

1. Загрузите эти файлы в GitHub-репозиторий.
2. В Render создайте новый Web Service из этого репозитория.
3. Если Render подхватит `render.yaml`, основные настройки применятся автоматически.
4. В Environment добавьте переменную `TMDB_API_KEY` со своим ключом TMDB.
5. Build command: `npm install`.
6. Start command: `npm start`.

Если ключ не настроен, сайт откроется, но живой рандом из TMDB будет недоступен.

## Важно про ключ TMDB

В старой HTML-версии ключ был прописан прямо в браузерном коде. В этой версии ключ хранится только на сервере через переменную окружения `TMDB_API_KEY`, поэтому его можно безопаснее держать вне GitHub.
